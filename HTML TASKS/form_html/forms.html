What is HTML?

HTML stands for HyperText Markup Language. It is the standard language for creating web pages and web applications. HTML describes the structure of a web page using markup. It consists of a series of elements that tell the browser how to display the content.

HTML elements are represented by tags. Tags are enclosed in angle brackets, and they usually come in pairs: an opening tag and a closing tag. The content goes between the opening and closing tags.
